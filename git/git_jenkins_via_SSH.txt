Как подключить дженкинс к репе в гите по ссх

1. Генерируем ключи ССХ с машины на которой стоит Jenkins, или с которой будем подклаться к гиту

2. Указываем открытый ключ в гите, settings - ssh and gpg keys - ssh keys

3. Создаем Job, в настройках указываем ссылку на репу, и креды - add - ssh username with private key - private key - указываем закрытый ключ

4. Прописываем ID
